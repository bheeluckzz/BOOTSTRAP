Apa itu HTML?

Jadi, HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web. HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah web.

CSS

Cascading Style Sheets (CSS) adalah bahasa style sheet yang digunakan untuk menentukan presentasi dan gaya dokumen yang ditulis dalam bahasa markup seperti HTML atau XML (termasuk dialek XML seperti SVG, MathML atau XHTML). CSS adalah teknologi landasan World Wide Web, bersama HTML dan javascript.

Bootstrap

Bootstrap adalah kerangka kerja CSS gratis dan open sour yang diarahkan pada pengembangan web front end mobile first yang responsif. Ini berisi HTML, CSS dan (opsional) template desain berbasis Javascript untuk tipografi, formulir, tombol, navigasi, dan komponen antarmuka lainnya.
